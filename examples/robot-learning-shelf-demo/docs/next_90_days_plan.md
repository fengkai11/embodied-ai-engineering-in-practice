# 后续 90 天学习计划（Chapter 20）

## 第 1 阶段：补齐感知与操作（1–30 天）
- 将当前状态特征扩展到多视角图像输入；
- 学习 6D pose estimation、grasp pose generation 与 basic manipulation；
- 完成货架异常检测 demo 的图像化输入版本。

## 第 2 阶段：补齐数据与学习（31–60 天）
- 扩展 teleop 数据采集方式；
- 引入更丰富的场景随机化；
- 把 ACT-like baseline 升级为更接近真实策略网络的版本；
- 引入 test 集和更完整的 rollout 评测。

## 第 3 阶段：补齐系统与部署（61–90 天）
- 补齐真实 ROS2 节点联调；
- 接入 MoveIt2 / 真机或仿真执行；
- 将桌面任务扩展到货架商品扶正和补货小闭环；
- 整理 GitHub README、视频与实验报告，用于求职与面试。
